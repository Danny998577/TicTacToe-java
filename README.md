# 🎮 Tic Tac Toe Game (Java)

A simple and interactive Tic-Tac-Toe game built using Java. This project includes both a console-based version and a GUI version using Java Swing.

---

## 🚀 Features

- 🧠 Two-player game (X vs O)
- 💻 Console-based version
- 🖱️ GUI version using Java Swing
- ✅ Input validation
- 🏆 Winner detection
- 🤝 Draw detection
- 🔄 Restart functionality (GUI)

---

## 🛠️ Technologies Used

- Java
- Java Swing (for GUI)
- OOP Concepts
